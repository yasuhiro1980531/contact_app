# アプリケーション名
お問い合わせフォーム、お問い合わせ内容管理システム

# 作成した目的
ユーザーから意見をもらい、その意見を管理し業務改善に役立てるため

# アプリケーションURL

# 機能一覧
お問い合わせフォームから送信された内容をDBに保存
管理システムでは名前、性別、登録日、メールアドレスのいずれかを入力すれば、ご意見を検索できる

# 使用技術
Laravel 8.x
